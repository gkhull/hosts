Welcome to Grant's Minhosts Hosts File Solution

This repository contains my personal hosts files.

Pick an option sorted by most to least protection.

OPTION 1: Use blacklist.txt, the blacklist in conjunction with annoyances.txt the annoyances list, and whitelist.txt, the whitelist.

OPTION 2: use greylist.txt if you cannot whitelist for any reason

The format is ||example.com^ to block all subdomains including the root

Blacklist Link: blacklist.txt

Whitelist Link: whitelist.txt

Greylist Link: greylist.txt

Annoyances Link: annoyances.txt

Feel free to use doh.txt to block DOH domains. This won't prevent anyone from hardcoding but i've found it less common for applications to hardcode the IP instead of using the domain.
